##  ⬇️ Installation ⬇️



⚠️ You must have Node.js installed on your system ⚠️

Install my-project with npm
 1. Go to the project folder
```bash
  npm install 
```
2. Install all dependencies (only once)
```bash
  npm install 
```
 3. Start the development server
 ```bash
  npm run dev
 ```
